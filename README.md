# 2.1-Shallow-Neuronal-Networks
This repository contains a collection of educational exercises focused on understanding shallow neural networks from first principles using Python and NumPy.

The objective is to develop an intuitive understanding of how neural networks operate, including linear transformations, activation functions, hidden layers, multiple inputs, multiple outputs, and loss functions.

# Topics Covered
## Linear Models
- One-dimensional linear functions
- Two-dimensional linear functions 
- Linearity in inputs and parameters
## Numerical Differentiation
- Central difference approximation
- Numerical derivatives
- Effect of the step size parameter
## Activation Functions
- Rectified Linear Unit (ReLU)
- Piecewise linear behavior
## Shallow Neural Networks
- One input, one output, three hidden units
- Two inputs, one output, three hidden units
- Two inputs, two outputs, three hidden units
- Hidden layer activations
- Output layer combinations
## Loss Functions
- Least-squares loss
- Manual parameter tuning
- Model fitting
